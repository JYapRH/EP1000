Arduino

We learnt about what electricity is as well as electrical components. We also learnt about how to use a breadboard which is a construction base for building electronics. It is used in many prototyping projects and is used for the quick testing of circuits.

We were tasked to create a 555 timer in TinkerCAD and simulate it. The 555 timer is configured as an astable using R and C values and the result is and alternating ON/OFF signal which flashes the LED.

Here is my simulated 555 timer in TinkerCAD

We also learnt about Embedded Systems. Embedded Systems are a dedicated computer system with a dedicated function within a larger system to perform a specific task. E.g. smart TVs, ovens. We will be working with Arduino Systems. It has open source libraries and support. It is inexpensive as there are lots of clones and 3rd party manufacturers. It has a simple, clear programming environment using GUI and is also cross-platform.

Next, we learnt about how to code with Arduino. Arduino code is loosely based on C++. A digital I/O pin can input or output digital (0, 5v) signals. The Arduino system provides 3 functions for the manipulation of digital I/O. We were also taught about the Analog I/O. Analog signals need to be converted to digital values before they can be processed.

Assignment

I was tasked to create a circuit whereby I have one switch that was able to control 3 LEDS and how they lit up. Initially, all of the LEDs are off. When the switch is pressed, the red LED turns on. When it is pressed again, the green LED turns on. When pressed again, the yellow LED turns on. When pressed again, all of the LEDs turn on. When pressed again, the LEDs flash between red, green then yellow. Finally, when it is pressed again, all of the LEDs turn off and they return to the first state

// Processing code // #define SW 2 #define GRE 13 #define RED 12 #define YEL 11 int count = 0; // Initial count value int state = HIGH; // Button not pressed void setup() { Serial.begin(9600); Serial.println("Start"); pinMode(SW, INPUT); pinMode(GRE, OUTPUT); pinMode(RED, OUTPUT); pinMode(YEL, OUTPUT); decode(count); } void loop() { if(digitalRead(SW) == LOW){ // Switch is pressed if (state == HIGH){ // Process state = LOW; } else { // do nothing } } else { if (state == LOW){ // Registers the keypress count = (count + 1)%6; decode(count); state = HIGH; } } } // Decodes count into binary void decode(int v) { switch(v){ case 0: // LEDs all off digitalWrite(RED, 0); digitalWrite(GRE, 0); digitalWrite(YEL, 0); break; case 1: // Only red LED on digitalWrite(RED, 1); digitalWrite(GRE, 0); digitalWrite(YEL, 0); break; case 2: // Only green LED on digitalWrite(RED, 0); digitalWrite(GRE, 1); digitalWrite(YEL, 0); break; case 3: // Only yellow LED on digitalWrite(RED, 0); digitalWrite(GRE, 0); digitalWrite(YEL, 1); break; case 4: // All LEDs on digitalWrite(RED, 1); digitalWrite(GRE, 1); digitalWrite(YEL, 1); break; case 5: // LEDs flash green then red then yellow digitalWrite(RED, 1); digitalWrite(GRE, 0); digitalWrite(YEL, 0); delay (500); digitalWrite(RED, 0); digitalWrite(GRE, 1); digitalWrite(YEL, 0); delay (500); digitalWrite(RED, 0); digitalWrite(GRE, 0); digitalWrite(YEL, 1); delay (500); break; } }
