Laser Cutting

Computer Controlled Cutting performs cutting operations based on digital data which is provided from CADD operations and digital 2D designs. It provides accurate and precise cutting that can be repeated. We learned about the process of laser cutting and designed a name tag for us to cut out of acrylic using the laser cutter at FabLab.

Step 1: Create a 2D design using Fusion 360 or other CAD software and export it into a file like DXF

Step 2: Import the file into CorelDraw which is provided with the laser cutter that you are using. CorelDraw allows us to add words onto our design that we can engrave

Step 3: Send the file to the laser cutter
